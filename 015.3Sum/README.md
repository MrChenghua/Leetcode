先将list排序，遍历list，i为当前遍历坐标，对i之后的点进行检索。
因为是求唯一解，如果nums[i]==nums[i-1]，则无需对当前坐标检索，因为之前已经对nums[i-1]的值进行过计算，直接continue。

对i之后的点进行检索时，取左坐标为l，右坐标为r，l为i+1,r为list的尾坐标。
在l<r的情况下，如果3数之和小于0，则左坐标加一；如果和大于0，则使右坐标-1。（因为数组已经排序，向右遍历数值增大，向左数值减小）

如果和为零，则将3个数传入list中，并且继续双向检索：l+=1，r-=1。
这里添加两个重复值检索，如果nums[l]==nums[l+1],则l+=1，如果nums[r]==nums[r-1],则r-=1。这里需要保证l<r。
